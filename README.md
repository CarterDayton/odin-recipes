# Odin Recipes

This project is a basic multi-page recipe website built as part of The Odin Project Foundations course. The goal of the assignment was to practice writing structured HTML and organizing a small website using proper file structure.

## Project Description

The website includes:

- A homepage (index.html) with navigation to individual recipes
- A recipes folder containing separate HTML pages
- An Images folder storing PNG files for each recipe
- Buttons and links to move between pages

Each recipe page contains:

- A title
- An image of the dish
- A list of ingredients
- Step-by-step instructions

## Skills Demonstrated

- Writing semantic HTML (headings, lists, links, images)
- Using relative file paths
- Organizing a multi-page project structure
- Basic Git workflow (add, commit, push)

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.
4. Use the buttons to navigate between recipe pages.
